VIA PHONE:
- First plug in your android device to your computer 
- Then drag the apk provided in my GitHub onto the filesystem on the phone via file explorer
- Then on your android device you should be able to find when you put it and open it
- Then the APK will install the app will open.

VIA Andriod Studio:
- First, open Andriod Studio
- Then open any project and go to tools->AVD manager
- Then if you don't already have a virtual device create a new one
- Once that is complete open the virtual device
- Then all you need to do is drag the apk onto the screen of the virtual device and open the app
